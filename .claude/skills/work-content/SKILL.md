# 作品詳細コンテンツの追記手順

## 絶対条件
- 一次情報（公式サイト・出版社ページ等）で確認できた事実だけを書く。創作・推測は禁止。
- 確認できない作品はファイルを作らない（省略表示が仕様）。
- 権利者の画像は使わない（サムネは AI 独断解釈の無関係イラスト＋「本作品との関連性はありません」注釈が既定）。

## 情報源の当たり順（段階チェックリスト）
1. 公式サイトのサブページ（story / introduction / about）
2. JS 描画で本文が取れないサイトは jina.ai リーダー経由で再取得
3. dアニメストアの作品ページ
4. eiga.com / animatetimes.com の作品紹介
※ どの段階で確認できたかをメモしておく（docs/operations.md「⑧作品詳細コンテンツの追記」の様式に従う）

## 手順
1. 対象作品の annictId を確認（/api/season の応答か content/works/index.ts）
2. 上記の順で一次情報を取得し、あらすじ・見どころ・出版社を確認
3. content/works/{annictId}.json を既存ファイルの形式に合わせて作成
4. content/works/index.ts に登録
5. dev で該当作品ページを表示確認（注意: dev 起動中に build を実行しない — キャッシュ破損の既知問題）
6. コミットは日本語1行 subject（例: 「作品詳細を3件追記」）。push は指示があった時のみ
