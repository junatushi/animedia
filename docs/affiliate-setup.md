# アフィリエイト運用手順

2026-07-18導入、2026-07-19に配信バッジへのリンク埋め込み方式に変更。
VODアフィリエイトの提携〜リンク登録〜月次更新の手順。
実装の全体像: サービスごとに複数ASPのリンクと報酬額をデータ登録し、**配信バッジ自体の
リンク先**として、その時点で報酬額が最大のactiveなリンクを自動採用する（コード変更不要）。
アフィリエイトが無いサービスのバッジは自動で公式サイトにリンクする（フォールバック）。

## 仕組み（どこに何があるか）

| 役割 | 実体 |
|---|---|
| リンク・報酬額のデータ（人力更新） | `content/affiliate/programs.ts` |
| 最高報酬リンクの自動選択 | `lib/affiliate.ts` の `pickAffiliate` |
| 公式サイトURL（フォールバック先） | `lib/services.ts` の `SERVICES[].officialUrl` |
| バッジのリンク化＋PR開示文の表示 | `components/ServiceMarks.tsx` |
| 表示箇所 | 配信バッジが出る全ページ（作品個別・シーズン一覧カード・サービス別） |
| クリック計測 | `affiliate_click`（アフィリエイト時）/ `official_link_click`（公式サイト時）。`/admin/analytics` で確認 |
| 開示ページ | `/privacy`（広告掲載方針・ステマ規制対応） |

**バッジの表示件数はアフィリエイトの有無で変わらない**（2026-07-19方針）。
アフィリエイトが無いサービスも従来どおりバッジとして表示され、リンク先が公式サイトになるだけ。
`content/affiliate/programs.ts` が未登録・全て `active: false` でも、
全バッジが公式サイトへのリンクとして機能する（＝提携前にデプロイしても安全）。

## 初回セットアップ（ユーザー作業）

1. ASPに登録する（無料）: A8.net・afb・バリューコマース・アクセストレード＋Amazonアソシエイト。
2. 各ASPでVOD案件（U-NEXT・DMM TV・ABEMA等）に提携申請する。
   審査には運営者情報の記載が必要（`/about` 整備済み）。Leminoはドコモアフィリエイトのみ。
   Netflixはアフィリエイト提携不可（2026-07調査）。
3. 承認されたら管理画面でリンクを発行し、`content/affiliate/programs.ts` に
   `{ asp, url, rewardYen, rewardNote, confirmedDate, active: true }` を追記する
   （ファイル内の記入例参照）。Claudeに「〇〇のアフィリエイトリンクを登録して」と
   依頼し、リンクと報酬額を貼り付ければ追記・検証まで行う。

## 月次更新（5分。ここだけは自動化できない）

報酬額はASP管理画面（ログイン制）でしか確認できないため、月1回:

1. 各ASPの管理画面で現在の報酬額を確認する。
2. 変わっていたら `programs.ts` の `rewardYen` と `confirmedDate` を更新する
   （Claudeに数字を伝えれば更新する）。**更新すれば採用リンクの切替は自動**。
3. 案件停止・提携解除されたものは `active: false` にする。

## 規約・法令上の注意（2026-07調査の要点）

- **PR表記**: アフィリエイトリンクになっているバッジには小さな「PR」タグが付き
  （`.svc-chip-pr`）、1件でもあればページに開示文（`.svc-disclosure`）が出る
  （消費者庁ステマ規制Q&A Q13: 冒頭の一括表記だけでは不十分とされる）。カード一覧のように
  同じバッジが繰り返される画面では`hideDisclosure`で開示文の重複だけ抑制するが、
  **PRタグ自体は省略しない**（バッジ単体でも広告と分かる状態を保つ）。
- **ロゴ・素材**: VOD各社はASP提供素材の権利を自社に留保し、画面キャプチャや
  ドメインへの商標使用を禁じている場合がある。本実装はテキストバッジ方式のため
  ロゴ問題を回避しているが、バナー掲載に変える場合は各社ガイドラインを確認する。
- **リンク属性**: アフィリエイトリンクは `rel="sponsored nofollow"`、公式サイトへの
  フォールバックリンクは `rel="noopener noreferrer"`（実装済み）。
- 成果条件（例: Leminoは32日間継続で成果確定）は `rewardNote` に記録しておく。
- **公式サイトURL** (`SERVICES[].officialUrl`) は各社トップページのみ（2026-07-18時点で
  一次情報確認済み）。ドメイン変更等に気づいたら都度更新する。
