## 流入最大化リサーチと着手候補（2026-07-13）

「サイトへの流入を増やす施策を出せるだけ出し、現実性を精査する」依頼を受けての整理。
施策の全リスト・却下理由の詳細はこのセッションの会話ログにあるが、要点と着手候補をここに残す。

### 調査方法と限界（正直に）
- WebSearch（US拠点・日本語検索ボリューム数値は取得不可）で、**競合が各クエリ族向けにページを
  作り込んでいるか**を需要の代理指標とした（競合はユーザーが検索する語に最適化するため）。
- したがって以下の「需要」は**検索ボリュームの実数ではなく、競合ページの厚みから読んだ推定**。
  正確な数値が要るなら、Google Search Console の検索パフォーマンス（実クエリ・表示回数）を
  数ヶ月ぶん見るのが一次情報として最も確実（既に登録済み）。

### 検証済みの需要（クエリ族ごと）
- ◎◎ **「独占配信」まとめ** — 競合が各クール専用ページを量産。ABEMAQuick「2026夏アニメ独占配信リスト」、
  anip.fun「独占配信まとめ」、kawareiworld、s-kaikibu、riopon「Netflixしか見れないアニメ全一覧」、
  さらに gamebiz・Yahoo!ニュース（オタク総研）まで記事化。**需要が最も明確**。
- ◎ **「[配信サービス] 今期アニメ 一覧」** — uzurea「dアニメストア 2026夏 配信作品一覧」、
  せにろぐ「ABEMAで見れる作品まとめ」、公式(dアニメ/ABEMA/Netflix)のシーズンページ。サービス軸の需要は確立。
- ○ **「今期アニメ 配信 どこで見れる 比較」** — VODコンパス・アニしま・ズッカズの森・アニメBB等が密集（激戦区）。
  本サイトのシーズンページが既に受けている領域。
- ○ **「声優 出演 今期アニメ」** — アニメイトタイムズが毎クール「声優別まとめ」を公開しX告知、
  seiyu.ie-t.net「声優共演検索」も存在。需要はあるが競合が強い。
- ○ **データ集計・ランキング系**（覇権ランキング・最速/先行配信の集計）— media も記事化しており拡散価値あり。

### 着手候補（優先順位つき）
1. [x] **★★★ 独占配信ページ `/exclusive/[year]/[season]`**（2026-07-13完了） — 需要◎◎ かつ独占チップ
   （見放題1社のみ判定）と同じ基準をSSRページ化。競合の「独占」は2〜3社独占を含む緩い定義が多いのに

2. **★★★ 配信サービス別ページ `/service/[key]`（新規）** — 需要◎。既存の`getSeasonData`＋`SERVICES`を
   そのまま再利用でき、「dアニメ 今期」等の受け皿・内部リンク増になる。実装コスト小・効果中〜大。
3. **★★☆ データ集計コンテンツ**（今期の配信サービス勢力図・最速配信ランキング等）— 需要○。Annictデータと
   `startedAt`比較で集計可能。権利画像不要で"データもの"として拡散・被リンク源になり、下記PR記事のネタにもなる。
4. **★★☆ 個人開発PR＋はてブ狙いの単発記事** — 一度きりだが初速が大きい。上記1〜3が揃ってから打つと着地率↑。
   ※日本のはてブ動向はUS検索では未検証。実行判断は別途。
5. **★☆☆ 声優別ページ `/person/[name]`** — 需要○だが競合（アニメイトタイムズ等）が強く、薄いページ量産で
   低品質判定される逆リスクあり。やるなら出演数上位に絞る。優先度は1〜3の後。

### 却下（非現実的）と理由
- ✕ **動画SNS（TikTok/Shorts/Reels/Instagram）** — 本サイトは権利画像を使わない方針（サムネすら無関係な創作）。
  映像・キャラ画像が命の動画SNSとは素材面で戦えない。制作コストも過大。
- ✕ **実況スレ/Reddit/海外流入** — 自己宣伝規制・炎上リスク。海外は「国内配信特化」と客層がズレて直帰しやすい。
- ✕ **プレスリリース（PR TIMES等）** — 1本3万円前後で個人サイトの費用対効果に合わない。同じ話題化なら無料の
  はてブ狙いが現実的。
- △ **LINE公式アカウント** — リテンションは強いが運用が重く、既存の配信通知メールと役割が重複。今の規模では過剰投資。
