## サイト改良（UX・機能）

### ★★★ 小さい労力で満足度が上がる
- [x] **PWA化**（2026-07-06完了）: `app/manifest.ts`＋`app/apple-icon.tsx`を追加。`app/icon.svg`も現行デザイン（ダーク×シアン）に更新。スマホの「ホーム画面に追加」に対応
- [x] **ライトモード切替**（2026-07-06完了、キャラクターモチーフで再設計）: ダーク＝「黒の剣士キリト」（既定の黒基調）、ライト＝「閃光のアスナ」（白×緋色×金、ナイツ・オブ・ブラッド意匠）。さらに剣戟の閃光（ヘッダーの光の筋・カードホバー時の斬撃）は土台と反転させ、ダークでは緋色×金の一閃（アスナ）、ライトでは漆黒の一閃（キリト）にしている。実装は`globals.css`にRGB三つ組の変数（`--accent-rgb`等）を導入し、`[data-theme="light"]`で一括差し替え。`components/ThemeToggle.tsx`で切替・localStorage保存
- [x] **モバイルのタップ領域見直し**（2026-07-07完了）: フィルターチップ・並び替え/シーズンタブ・年プルダウン・ヘッダーの共有/テーマボタン・カード右上のアクションボタンを、480px以下で44px前後（Apple/Google推奨の最小タップサイズ）まで拡大。隣接ボタン間の間隔も広げて誤タップを防止

### ★★☆ 中規模だが価値が高い
- [x] **作品単位の動的OGP**（2026-07-06完了）: `app/anime/[id]/opengraph-image.tsx`を追加。作品名・配信サービス一覧入りのシェア画像を生成（`lib/ogFont.ts`にフォント読み込みを共通化）
- [x] **配信スケジュールカレンダー表示**（2026-07-07完了）: 一覧画面に「一覧/カレンダー」の表示切替を追加。Annictのprograms（放送/配信ごとの記録）が持つ`startedAt`から、作品ごとに最も早い日時を求めてJSTの曜日・時刻を算出し、月〜日＋「配信日未定」の8列で表示。検索・絞り込みと連動する。時刻は初回の実績からの推定であることを注記（話数によるズレはAnnict側の性質上あり得るため）
- **通知/リマインダー機能**: お気に入り作品の配信開始をお知らせ。ただし現状の「DBなし・localStorageのみ」という設計方針とは相性が悪く、要実現方法の検討（メール送信基盤や外部通知サービスが必要になり、コストも絡む）

### ★☆☆ 検討要（既存方針との整合が必要）
- 声優・スタッフでの絞り込み（Annictのデータ範囲次第）
- アクセシビリティ改善（スクリーンリーダー対応、コントラスト比の再チェック）
