# 見込みユーザー・コミュニティ候補ログ

`outreach-scout` サブエージェント（`.claude/agents/outreach-scout.md`）が実行のたびに追記する累積ログ。
Web検索・WebFetchで見つけた「Discordサーバー」「まとめ/感想サイト」「配信どこ系の困りごと」「Xの公開言及」の
候補を、出典URLつきで一覧化したもの。**このファイルはあくまで候補一覧であり、接触・投稿・DM送信は行わない**
（`docs/sns-growth-research.md` の方針どおり、実際の交流・掲載依頼は人力で行う）。

使い方: 気になる候補があれば、実際にDiscordサーバーに参加して交流する／記事へのコメントや掲載依頼を
手動で送る、といった人力アクションにつなげる。自己宣伝を規制しているコミュニティが多いため、
いきなりURLを貼らず自然な参加から始めること。

起動タイミングの目安: 新クール開始時・SNS施策を見直したいタイミングで随時（`docs/operations.md` 参照）。
